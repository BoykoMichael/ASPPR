Необхідно описати структуру програми, вхідні й вихідні дані, навести екранні форми (screenshots) програми, а також продемонструвати виконання як тестових прикладів, так і власного варіанта завдання.

-x1 + 3x3 – 2x4 + x5 ≤ 3;

ВИКОНАННЯ РОБОТИ
Для виконання практичної роботи було розроблено консольний застосунок мовою C#, призначений для знаходження опорного і оптимального розв’язків задачі лінійного програмування. Програма передбачає введення кількості змінних, кількості обмежень, типу цільової функції (min/max), Z-рядку і рівняння обмежень із подальшим виведенням результату з відповідним математичним рішенням разом із детальним протоколом обчислень. Структура системи представлені на діаграмі класів (рис. 2), де клас Program є основним класом який визиває методи з інших класів. Клас InputHandler який включає методи для обробки вводу. Клас SomplexSolver який включає усі математичні алгоритми. Клас ReportPrinter, призначений для створення протоколу обчислень.

Протоколи обчислень для розрахунків оберненої матриці та системи рівнянь відповідно до варіанту 5 можна побачити на рисунку 3. Лістинг програми можна побачити у додатку А. До репозиторію GitHub з проєктом можна перейти за посиланням.

ВИСНОВКИ
Під час виконання роботи було опановано методику розв’язання задач лінійного програмування за допомогою модифікованих жорданових виключень (МЖВ) та реалізовано відповідний програмний комплекс мовою C#. Розроблений застосунок дозволяє автоматизувати двоетапний процес пошуку розв’язку: спочатку – знаходження опорного плану (фаза 1), а потім – визначення оптимального значення цільової функції (фаза 2). Тестування програми на власному варіанті №5 та на контрольному прикладі з необмеженою цільовою функцією підтвердило коректність роботи математичного алгоритму, правильність формування симплекс-таблиць та здатність системи адекватно обробляти різні типи результатів лінійного програмування.
