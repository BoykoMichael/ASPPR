
Мета роботи – Навчитися застосовувати звичайні жорданові виключення (ЗЖВ)

ПОСТАНОВКА ЗАДАЧІ

Під час виконання практичної роботи необхідно розробити програму, за допомогою якої на основі звичайних жорданових виключень можна реалізувати такі дії:
1.	Пошук оберненої матриці для довільної квадратної матриці.
2.	Обчислення рангу довільної прямокутної матриці.
3.	Розв'язання системи лінійних алгебраїчних рівнянь за допомогою оберненої матриці (1-й спосіб).

ВАРІАНТ 5

ВИКОНАННЯ РОБОТИ

Для виконання практичної роботи було розроблено консольний застосунок мовою C#, призначений для знаходження обернених матриць, обчислення їхнього рангу та розв’язання систем лінійних алгебраїчних рівнянь за допомогою оберненої матриці (1 метод) на основі методу Жорданових виключень. Програма передбачає введення вхідних матриць із подальшим виведенням результату з відповідним математичним рішенням разом із детальним протоколом обчислень. Структура системи представлені на діаграмі класів (рис. 2), основним і єдиним класом є клас Program.

Протоколи обчислень для розрахунків оберненої матриці та системи рівнянь відповідно до варіанту можна побачити у додатку А. Лістинг програми можна побачити у додатку Б. До репозиторію GitHub з проєктом можна перейти за посиланням: https://github.com/BoykoMichael/ASPPR/tree/master/ASPPR%20LAB1

ВИСНОВКИ

У ході виконання лабораторної роботи було практично реалізовано алгоритм модифікованих жорданових виключень мовою C# для автоматизації задач лінійної алгебри. Розроблений програмний інструмент дозволяє коректно визначати ранг матриць довільної розмірності, обчислювати обернену матрицю для квадратних структур шляхом послідовних кроків ЗЖВ та знаходити розв'язки систем лінійних алгебраїчних рівнянь методом матричної інверсії. Отримані результати підтвердили ефективність ітераційного підходу Жордана-Гаусса для чисельних методів, а також дозволили вдосконалити навички розробки прикладного програмного забезпечення для обробки складних математичних структур і багатовимірних масивів.
